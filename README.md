A lightweight Identity Provider (IdP) focused on secure authentication and minimalist design.

---

### Core Features

- **Security**: Industrial-grade encryption and secure cookie handling.
- **UI**: Single-fold landing page built for speed and clarity.
- **Theming**: Native Dark and Light mode support.
- **Responsiveness**: Uses `h-dvh` to prevent vertical clipping on short screens.

---

### Tech Stack

| Layer        | Technologies                                          |
| :----------- | :---------------------------------------------------- |
| **Frontend** | React (Vite), Tailwind CSS, shadcn/ui, Sonner, Lucide |
| **Backend**  | Node.js, Express, Helmet, Cookie-parser               |
| **Database** | MongoDB (Mongoose), Redis (Caching/Sessions)          |
